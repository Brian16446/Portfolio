Book Tracker


About this project:

This is a project I created during the winter holidays of 2018. I used this project to familiarise myself with HTML/CSS/JS and PHP before I started my web coursework for the year.
I learned a lot about HTML and CSS from this project and learned how to pull data from a MySQL database. I also learned about POST and GET requests and how AJAX works.
The project could be improved upon as I had to stop working on it to start my coursework and so there are some features that I could have added, had I had the time. I would've liked to add the ability to search by rating and to edit information about a book such as the rating or date read. I also would've liked to sharpen up the searches a bit as if you search by author and then change the year, you have to retype your search into the search box. Also I'd have liked it if the background faded into the basic color from the image. I could go back and add these things in, but I feel like I learned a lot of PHP and web development this year so I'd rather learn something new than go back over old projects to brush them up.
